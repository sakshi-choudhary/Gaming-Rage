Jump Game

A game where user can jump on obstacle based upon memory reaction and score +1 for every obstacle cross.

Instructions!

  • - Obstacle are red in color
  • - Speed remains constant throughout
  • - Jump through obstacle for +1 score
  • - right click for jumping accordingly
  • - if you hit the obstacle its a game over for you.
  • - Score will be shown on top after game is over.
  • - HAPPY GAMING!!
